## Installation

1. install nodejs (node v16.x + npm v7.x OR node v18.x + npm v8.x) for your platform
2. go to project's directory
3. run `npm i`


## Dev mode

1. make installation
2. go to project's directory
3. run `npm run front:start-*` and\or `npm run back:start-*`
